## 💡 About this tool

Temper-domains helps you find available web domain names. It runs directly inside your computer terminal. You enter a name, and the tool checks if someone already owns it. This process happens fast. The tool protects your privacy because it does not track your searches. You can use it to find the perfect name for a new project or business website.

## 🖥️ System requirements

This tool works on Windows 10 and Windows 11. Your computer requires a standard internet connection to contact domain name servers. You do not need to install complex programming runtimes to use the application. The program contains everything it needs to run on its own.

After you download the file, you must run it through the Windows terminal. Follow these instructions to start your search:

1. Open your "Downloads" folder.
2. Hold the "Shift" key on your keyboard and right-click on an empty space inside the folder.
3. Select "Open PowerShell window here" or "Open in Terminal" from the menu.
4. Type `.\temper-domains.exe` and press the "Enter" key on your keyboard.
5. The application will open inside the window.

## 🔎 How to search for domains

Once the tool opens, you see a blinking cursor. This is the command line interface. Type the name you want to check for your website. If you want to check for "mycoolwebsite", type that word and press "Enter". 

The tool will reach out to central databases to see if the name is available. You will see a message on your screen telling you if the name is taken or free for you to register.

## 📑 Understanding the results

The tool provides simple output messages. 

- "Available": This means nobody owns the name. You can visit a domain registrar to purchase it.
- "Taken": This means someone already owns the domain name. You should try a different name or add an extra word to your search.
- "Error": This happens if the connection to the domain database times out. Wait five seconds and try your search again.
